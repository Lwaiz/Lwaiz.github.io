你好，我是 **Leo**，欢迎来到我的个人主页。


## ❤️ 初衷

本人一直是 [FlowUs 息流](https://flowus.cn/product) 的重度用户，由于它能手机、Pad、电脑多端同步。但是一直对其云端存储（没有本地存储）耿耿于怀，总觉得不安全，生怕哪天数据全没了。

后来想搭建一个博客网站，最初的原因其实很简单：
希望能把自己在 FlowUs 上写的内容，有一个长期、稳定的备份。

一开始，因为一直没找到真正喜欢的博客模板，总觉得现成的主题差点意思，于是干脆自己动手，基于 GitHub Pages 写了一个博客 —— [My_Note](https://lwaiz.github.io/My_Note/index.html)。

不过慢慢发现，自己花了越来越多时间在折腾博客系统本身，而不是记录内容。

后来接触到了 [Hexo](https://hexo.io/zh-cn/) + [Butterfly](https://butterfly.js.org/)，发现原来博客也可以做到既简洁又美观。于是重新搭建了现在这个网站，也终于把更多精力重新放回了“写东西”这件事上。

后期，有机会再继续琢磨吧！
